Padding VS Border

Padding Border

e.g. original block size is 500x500 > added 10px padding to 4 sides of the block > block size is 520x520
 Border does not affect size of the block
e.g. original block size is 500x500 > added 10px border to 4 sides of the block > block size is still 500x500

Position

  1. position: relative
    • position relative to its normal position
    • option: top, left, right, botton
    • will not affect other elements's position > may overlap with other elements
    • think of adding margin to the element
  2. position: absolute
    • directly state the distance between the element and its RELATIVE parent
    • think of the element is positioning relative to the body, ignoring parent and other element
  3. position: fix
    • always stay on the defined position

Fonts

  • CSS font stack: https://www.cssfontstack.com/
  • 5 font families
    1. SERIF - serif
      • default font-family
    2. SANS-SERIF - sans-serif
      • compare to "sarif", it does not have "tails"
    3. MONOSPACE - monospace
      • every characters have same width
    4. CURSIVE - cursive
    5. FANTASY - fantasy
  • Each font family has different fonts
    • e.g. Monospaced has Consolas, Andale Mono, ...

Computer may be unable to show the selected font from the HTML, thus we have a FALL BACK system

  • Browser will try to show the font stated at first, if unable to show, use next fonts, and so on
  • e.g. font-family: Arial, Helvetica, sans-serif;
  • priority of showing the font: Arial > Helvetica > Any font from font-family "sans-serif"

Or we can use EMBED way to solve it

  1. Go to Google Fonts to check out fonts able to be embeed: https://fonts.google.com/
  2. Select font and copy code on right side
  3. Paste the code in head section
  4. Then able to use the added font in css font-family as usual

Dynamic Font Size

If font unit is "px", users may see different font size as stated in px if their default font size is not "medium"

  • em
    • set the font size in ratio of parent's font size
    • (without additional setting) 1em 100% = 16px
    • if you want the font size is 80px in normal ratio, you should: 80px/16px = 500% = 5em
    • then the text would keep same ratio even the accessibilily is enlarged / diminished
  • rem
    • set the font size in ratio of root's font size (i.e. 16px)
    • (must) 1em 100% = 16px
